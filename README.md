# Shelley's Entertainment Hub

## Upload
1. Open your `shelleys-entertainment-hub` repository.
2. Choose **Add file → Upload files**.
3. Upload all five files from this package.
4. Commit the files.
5. Open **Settings → Pages**.
6. Choose **Deploy from a branch**.
7. Select `main` and `/ (root)`.
8. Save and wait 1–3 minutes.

This starter version uses sample data from `data.js`. The next phase connects it to your Google Sheet through Apps Script so the website updates automatically.
